## Overview

QuickTasker is a lightweight and user-friendly task management tool designed to help individuals and teams organize, track, and complete tasks efficiently. With a clean interface and powerful features, QuickTasker makes managing your daily to-dos a breeze.

---

## Features

- Create, edit, and delete tasks effortlessly
- Organize tasks by projects and categories
- Set due dates and reminders
- Prioritize tasks with color-coded labels
- Mark tasks as complete and view progress
- Responsive design for desktop and mobile

---

## Tech Stack

- Frontend: React.js
- Backend: Node.js with Express
- Database: MongoDB
- Authentication: JWT
- Styling: Tailwind CSS

---

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/tammydidea74/QuickTasker.git
   ```

2. Navigate to the directory:

   ```bash
   cd QuickTasker
   ```

3. Install dependencies for both client and server:

   ```bash
   cd client && npm install
   cd ../server && npm install
   ```

4. Set up environment variables (create `.env` in server folder):

   ```
   MONGODB_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret
   ```

5. Run the development server:

   ```bash
   # From server folder
   npm start

   # In a separate terminal window, from client folder
   npm start
   ```

---

## Usage

- Open your browser and navigate to `http://localhost:3000`
- Sign up for a new account or login
- Start creating tasks and organizing your projects
- Use task labels and due dates to stay on top of your work
